Implementing a Ripla application

To create a web application with Ripla, you have to do the following steps:

  • Create an application bundle
  • Create various use case bundles
  • Create a skin bundle
  • Deploy and enjoy

Note: throughout this document, the term bundle is meant as OSGi bundle.
